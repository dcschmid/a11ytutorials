Blinde Menschen können Webseiten nicht so gut verstehen wie Menschen, die sehen können. Visuelle Benutzer können das Seitenlayout sehen
und verstehen, ohne den Text lesen zu müssen. Blinde Nutzer haben diesen Vorteil nicht. Screenreader lesen die Webseite Zeile für Zeile.
Sie hören sich die gesamte Webseite an, es sei denn, es gibt eine andere Möglichkeit, um zu verstehen, wie die Seite aufgebaut ist.
Überschriften sind eine Möglichkeit, das Layout einer Webseite zu verstehen.

Benutzer von Screenreadern können mit Tastenkombinationen durch die Überschriften eines Dokuments navigieren. Die meisten Screenreader
können folgende Aufgaben ausführen:

## Methode 1: Liste alle Überschriften auf

Einige Screenreader zeigen dies in einer Baumansicht an. Andere, wie zum Beispiel VoiceOver, zeigen alle Überschriften in einer Liste
mit der Nummer der Überschriftenebene an.

<Aside type="tip" title="Beispiel: Überschriftenliste in NVDA">
<Picture src={HeadingsJaws} formats={["avif", "webp"]} alt="" />

Die Überschriften können nach einer bestimmten Reihenfolge sortiert werden. Benutzer können auch auswählen, welche Überschriften
angezeigt werden sollen. Benutzer von Screenreadern können schnell die Überschriften durchgehen und verstehen dann, wo sie sich
befinden. Aber nur, wenn eine Webseite echte Überschriften hat und diese einen guten Überblick über den Inhalt geben, kann man
sie schnell verstehen.

</Aside>

<Aside type="tip" title="Beispiel: Überschriftenliste in JAWS">
<Picture src={NvdaHeadingsList} formats={["avif", "webp"]} alt="" />

NVDA zeigt die Überschriften in einer Baumansicht an. Die Zwischenüberschriften werden in einer Liste unter der jeweiligen
Überschrift angezeigt. Sie können die Listen verkleinern oder erweitern, indem Sie die Pfeiltaste nach links oder rechts
drücken, wenn eine Überschrift markiert ist, oder auf das Minus- oder Pluszeichen neben einer Überschrift klicken.

<Picture src={NvdaHeadingsListFiltered} formats={["avif", "webp"]} alt="" />

Man kann die Liste so filtern, dass nur die Überschriften angezeigt werden, die die
Textzeichenfolge enthalten, die man in das Textfeld "Filtern nach:" eingibt.

</Aside>

## Methode 2: Navigieren Sie von einer Überschrift zur nächsten.
