## Sie können alle Elemente auswählen, die Sie fokussieren möchten.

Die Tab-Taste ist die einfachste Methode, um durch Links auf einer Seite zu navigieren. Dafür drückst du einfach die
Tab-Taste (oder Shift + Tab, um rückwärts zu gehen). So kannst du schnell von einem Link zum nächsten springen. Wenn
Sie nur an Links interessiert sind, ist das nicht die beste Methode. Aber sie funktioniert, wenn Sie ein paar zusätzliche
Tastenanschläge verwenden möchten.

## Navigieren Sie von einem Link zum nächsten

<table>
  <thead>
    <tr>
      <th>Bildschirmlesegerät & Browser</th>
      <th>Befehl</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>JAWS mit Chrome, Firefox, IE</td>
      <td>Nicht verfügbar</td>
    </tr>
    <tr>
      <td>NVDA mit Firefox, Chrome</td>
      <td>K</td>
    </tr>
    <tr>
      <td>VoiceOver mit Safari (iOS)</td>
      <td>
        Verwenden Sie den Rotor, um Links auszuwählen (drehen Sie zwei Finger
        auf dem Bildschirm nach links oder rechts) und wischen Sie dann mit
        einem Finger nach unten
      </td>
    </tr>
    <tr>
      <td>Talkback mit Firefox</td>
      <td>
        Lokales Kontextmenü (wischen Sie nach oben, dann nach rechts) und wählen
        Sie dann Links aus
      </td>
    </tr>
    <tr>
      <td>VoiceOver mit Safari (macOS)</td>
      <td>Strg + Option + Cmd + L</td>
    </tr>
    <tr>
      <td>Narrator mit Edge </td>
      <td>K</td>
    </tr>
  </tbody>
</table>
