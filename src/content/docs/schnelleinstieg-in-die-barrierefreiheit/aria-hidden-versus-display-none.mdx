---
title: aria-hidden versus display:none
---

Man kann Elemente ausblenden, damit man sie nicht sieht. Das macht man, wenn man ein Menü hat, das man erweitern kann. Oder wenn man
ein Menü hat, das in einer Baumansicht ist. Oder wenn man ein Menü hat, das nicht aktiv ist.

Das wird meistens mit der CSS-Eigenschaft "display: none" erreicht. Alles, was mit "display: none" markiert ist, ist nicht verfügbar.

Es gibt eine andere Eigenschaft namens "aria-hidden". Das Element wird nur für Benutzer von Hilfstechnologien ausgeblendet. Es bleibt
aber auf der Seite, damit sehende Benutzer es weiterhin sehen können. Das ist keine gute Idee. Manchmal ist aria-hidden eine gute
Lösung. Aber seien Sie vorsichtig!
