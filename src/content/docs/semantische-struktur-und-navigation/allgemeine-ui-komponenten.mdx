title: Allgemeine UI-Komponenten
---

## Bei Inhalten, die mit Auszeichnungssprachen umgesetzt werden, kann der Zweck von Benutzeroberflächenkomponenten, Symbolen und Bereichen programmgesteuert bestimmt werden.

Dieses Erfolgskriterium baut auf 1.3.5 auf. Damit kann man nicht nur den Zweck von Texten und Symbolen erkennen, sondern auch von
Bereichen einer Seite und anderen Elementen der Benutzeroberfläche. Der Seitencode enthält Informationen, die von anderen Programmen
gelesen werden können. So können zum Beispiel zusätzliche Angaben gemacht werden oder die Seite kann für den Benutzer geändert werden.

So können Benutzer ihre Web-Erfahrung anpassen. Benutzer, die Symbole lieber als Wörter nutzen, könnten mithilfe von Hilfstechnologien
benutzerdefinierte Symbole auf dem Bildschirm platzieren. Man könnte auch Bereiche der Seite ausblenden, die man nicht sehen will. Das
wäre gut für Menschen, die sich im Internet mit Symbolen orientieren, die ihnen gefallen. Das sind zum Beispiel Menschen, die nicht
gut lesen, verstehen oder sich konzentrieren können.

Das W3C weiß noch nicht, wie die zusätzlichen Metadaten bereitgestellt werden sollen, damit die Anpassung möglich ist. Bleiben Sie
dran!
