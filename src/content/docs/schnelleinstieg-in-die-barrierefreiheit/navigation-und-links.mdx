## Navigation

Man kann zwischen Seiten navigieren. Man kann auch innerhalb einer Seite navigieren.

Die Navigationslinks sollten im Inneren eines Landmarks gesetzt werden. Das ist ein Punkt.

Wenn es mehrere Links hintereinander gibt, sollten sie im Listenformat aufgelistet werden. Sie können es auch so machen, dass es nicht
wie eine Liste aussieht. Aber es ist trotzdem eine Liste. Die Benutzer von Screenreadern können sagen, dass es eine Liste mit fünf
Elementen gibt. Sie können jedes Element einzeln durchgehen.

Wenn Sie Menüs mit Untermenüs haben, machen Sie eine Liste mit den wichtigsten Punkten und darunter eine Liste mit den Unterpunkten.

Jeder Navigationslink muss vorhanden sein. Der Linktext muss aussagekräftig sein. Er muss auf jeder Seite gleich sein. Wenn Sie auf
einer Seite "Kontaktieren Sie uns" schreiben, schreiben Sie auf einer anderen Seite nicht "Senden Sie uns eine E-Mail". Verwenden Sie
immer die gleiche Formulierung.

Wenn ein Link weiter nach unten oder über den aktuellen Standort hinausführt, sollte das Linkziel Text enthalten. Wenn Sie nur einen
leeren Behälter schicken, kann der Screenreader nichts sagen.

## Links

Hier sind ein paar Dinge, die Sie bei allen Links beachten sollten. Zum Beispiel, ob es sich um einen Navigationslink handelt, um auf
Ihrer Website zu navigieren, oder um Ihre Website zu verlassen.

1. Nummer eins, Links müssen mit der Tastatur fokussierbar sein, Das bedeutet, dass Sie in der Lage sein müssen, die Tabulatortaste zu
   verwenden , um auf einem Link zu landen.

2. Nummer zwei, sie müssen über eine visuelle Fokusanzeige verfügen Wenn Sie mit der Tabulatortaste zum Link navigieren.

3. Nummer 3, Links müssen visuell unterscheidbar sein Von Text, der kein Link ist. Und Sie können dies mit Unterstreichungen tun, Oder,
   wenn es sich in einem Navigationsmenü befindet, Es könnte einen Umriss drumherum haben, Oder etwas in dieser Richtung, Aber Links
   müssen anders aussehen als normaler Text.

4. Der Linktext muss aussagekräftig sein, Denn eines der Dinge, die Screenreader-Benutzer oft tun werden ziehen oft eine Liste von
   Links auf, Wo nur der Linktext angezeigt wird. Wenn also Ihr Link "hier klicken" oder "weiter" sagt, oder "mehr", oder "lies mehr",
   Das sind alles Phrasen, die etwas bedeuten, Aber nur im Kontext, wo sie vorkommen. Wenn Sie sie aus diesem Kontext extrahieren, sie
   ergeben nicht so viel Sinn, Es empfiehlt sich also, Text zu verwenden Die in eine Liste von Links extrahiert werden können.

5. Und der Link muss ein echter Link sein, So kann er von assistiven Technologien als solches erkannt werden. Das bedeutet, dass das
   `<a>` Tag mit einem gültigen href-Attribut verwendet wird, Oder, in selteneren Fällen, kann es auch bedeuten Verwendung des
   ARIA-Musters für Links, Dies erfordert zusätzliches Markup und Skripting. Normalerweise wird das nicht empfohlen, Weil ein Link ein
   Link ist, Und es ist immer besser, den einfachen Weg zu gehen, Weil es weniger anfällig für Fehler ist, während der Entwicklung.
