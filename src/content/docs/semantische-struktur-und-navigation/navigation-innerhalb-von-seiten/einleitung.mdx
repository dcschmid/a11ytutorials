---
title: Navigation innerhalb von Seiten
---

Mit Tastenkombinationen können Sie die meisten Elemente auf einer Webseite ansteuern, zum Beispiel Überschriften, Bilder, 
Tabellen und Links. Wenn Sie sich also mit der Struktur befassen, können Nutzer von Bildschirmleseprogrammen einfacher auf 
der Seite navigieren.

Es gibt aber auch Dinge, für die es in HTML noch keine guten Lösungen gibt. Es gibt keine Möglichkeit, in einer 
Suchergebnis-Ansicht mitzuteilen, dass man sich an der aktuellen Stelle befindet. In diesem Abschnitt zeigen wir, wie das geht.

Auch sehen Menschen, die eine Tastatur benutzen, nicht alle Tastaturkürzel in Bildschirmleseprogrammen. Deshalb können sie 
nur wenige Seiten auf diese Weise durchgehen. Man kann auch einen Link einfügen, mit dem man die Navigation überspringen kann.
