---
title: Werte - Statische Eigenschaften
---

Sie können auch andere Werte zuweisen, zum Beispiel mit ARIA-Elementen.

Sie können mit "aria-valuemin" und "aria-valuemax" einen Mindest- und einen Höchstwert festlegen, damit der Benutzer keine falschen
oder irrelevanten Werte auswählt.

Sie können ein ARIA-Datenraster mit Eigenschaften wie "aria-colcount" für die Anzahl der Spalten und "aria-haspopups" erstellen.
Bei einigen Elementen, z. B. Menüs und Web-Apps, können Sie die Benutzer informieren, dass ein Element ein Popup hat.

Sie können in einem Dialog "aria-modal=true" eingeben, damit Hilfstechnologien alle Elemente ignorieren, die nicht im Dialog
enthalten sind.

Alle bisher genannten Eigenschaften sind meistens immer gleich. Diese Werte bleiben meistens gleich.
