## Einführung und Einrichtung

Screenreader sind Programme, die Texte vorlesen oder in Brailleschrift umwandeln. So können auch Menschen mit Sehbehinderung, egal ob
blind oder taubblind, oder mit schlechtem Sehvermögen, auf Texte auf digitalen Geräten zugreifen. So können sie mit digitalen
Oberflächen arbeiten.

Es ist wichtig, Webseiten mit Screenreadern zu testen. So kann man sehen, ob sie barrierefrei sind. Wenn Sie noch nie einen Screenreader
benutzt haben, kann das einschüchternd sein.

Es gibt keine grafische Benutzeroberfläche. Es ist ein Gerät, mit dem Sie Ihren Computer über die Tastatur steuern können. Für die
meisten von uns ist das eine große Veränderung. Das ist in Ordnung. Dieses Video hilft Ihnen, den Übergang zu machen.

## Einen Screenreader auswählen

Welcher Screenreader am besten passt, hängt davon ab, welches Gerät Sie nutzen. Die meisten Benutzer von Desktop-Screenreadern nutzen
Windows mit dem NVDA- oder dem JAWS-Screenreader. NVDA und JAWS sind sehr verbreitet und können viel.

## Herunterladen und Installieren von NVDA

Laden Sie NVDA von NV Access herunter. Es ist kostenlos und Open-Source. Sie können aber auch spenden, wenn Sie möchten.
möchtest.

Während der Installation wählen Sie bitte die Option "Desktop-Symbol erstellen" und "Tastenkürzel erstellen". Das brauchen wir
später. Die Installation ist fast fertig. Ihr Bildschirmlesegerät ist dann bereit und kann Ihnen viel helfen.

Drücken Sie Strg, um zu beenden.

## Ein- und Ausschalten von NVDA

Um NVDA auszuschalten, drücken Sie Strg + Alt + N. Um es wieder einzuschalten, drücken Sie dieselbe Tastenkombination.

Wenn Sie die Tastenkombination vergessen, können Sie die NVDA-Startschaltfläche hinzufügen. So haben Sie einen einfachen Zugriff auf
Ihre Taskleiste. So können Sie es mit einem Klick einschalten. Wenn Sie NVDA mit einem Mausklick ausschalten möchten, gehen Sie zur
Systemablage, klicken Sie auf das NVDA-Symbol und wählen Sie "Beenden".

## Mausverfolgung deaktivieren

In der Standardkonfiguration von NVDA wird der Name von allem angezeigt, wenn Sie mit der Maus darüber fahren.

Das ist eine Hilfe für Menschen mit Sehbehinderung. Sie können die Funktion aktiv lassen, wenn Sie möchten. Die meisten sehenden
Tester für Barrierefreiheit schalten diese Funktion aus, damit der Screenreader leiser ist, wenn sie die Maus bewegen. Das müssen Sie
in den Einstellungen einstellen.

Klicken Sie auf das NVDA-Symbol in der Taskleiste. Klicken Sie auf "Voreinstellungen" und "Einstellungen". Oder drücken Sie "Einfg"
+ "N", um die Einstellungen mit der Tastatur zu öffnen.

Wählen Sie "Maus" und deaktivieren Sie die Option "Maus-Tracking aktivieren". Wenn Sie schon mal hier sind, schauen wir uns weitere
Optionen an.

## Ändern der Stimme

Die Standard-NVDA-Stimme klingt wie ein Roboter. Wenn Ihnen die Stimme nicht gefällt, können Sie sie in den Einstellungen ändern.
Drücken Sie auf "Ändern", um den Synthesizer zu ändern.

Wählen Sie die Windows-OneCore-Stimme aus.

Klicken Sie auf OK.

Probiere jetzt eine andere Stimme aus.

## Ändern Sie die Geschwindigkeit

Sie können die Geschwindigkeit des Screenreaders mit dem Geschwindigkeitsregler anpassen.

## Aktivieren Sie die Fokushervorhebung

Für sehende Tester ist es hilfreich, wenn sie die Funktion zum Hervorheben des Fokus aktivieren können.

Gehen Sie zu "Vision" und aktivieren Sie die visuelle Hervorhebung.

Ich empfehle, alle Optionen für die Fokushervorhebung zu aktivieren. Wenn die Fokushervorhebung aktiviert ist, sehen Sie immer, wo
der Screenreader liest. Sie werden es später sehr zu schätzen wissen.

## Ändern Sie die NVDA-Modifikatortaste

Die Standard-Taste zum Ändern in NVDA ist die Einfügetaste. Die Zusatztaste wird zusammen mit anderen Tasten verwendet, zum Beispiel
zusammen mit Einfg und F7. Damit kann man eine Liste mit Links, Überschriften, Formularfeldern, Schaltflächen und Orientierungspunkten
öffnen.

Wenn Ihre Tastatur keine Einfügetaste hat oder Sie sie nicht benutzen wollen, können Sie eine andere Taste als NVDA-Modifizierertaste
festlegen, zum Beispiel die Feststelltaste.

## Aktivieren der Sprachausgabe-Anzeige

Wenn Sie Webentwickler oder andere testen, können Sie die Ausgabe des Screenreaders im Textformat mit Sprachanzeige über das
NVDA-Symbol in der Taskleiste erfassen.

Wählen Sie "Werkzeuge", dann "Sprachanzeige".

Die Sprachanzeige zeigt an, was der Screenreader sagt. Sie können den Text kopieren und in Bug-Tracking-Tickets oder Nachrichten
einfügen.

Jetzt können Sie mit dem Testen beginnen.
