---
title: Sprache
---

Die meisten Bildschirmlesegeräte können mehrere Sprachen lesen. Bei der Installation wählen Benutzer eine Standardsprache aus.
Wenn im HTML-Dokument keine Sprache angegeben ist, liest das Bildschirmlesegerät das Dokument in der Sprache des Benutzers vor.
Wer nur eine Sprache spricht, wird nur in dieser Sprache auf Websites zugreifen. Benutzer, die mehr als eine Sprache sprechen,
werden auf Websites in mehr als einer Sprache zugreifen wollen. Sie möchten, dass jede Sprache korrekt vorgelesen wird.
